    ```astro

    const { title = "Bostrich" } = Astro.props;
    ---
    <html lang="en">
      <head>
        <meta charset="utf-8" />

        </main>
        <Footer />
      </body>
    </html>
    ```

    - Accepts optional title prop with default "Bostrich"
    - Uses semantic HTML: <header>, <main>, <footer>
    - Provides a slot for page content
    - Applies Tailwind classes for spacing and colors
    - Exports the Layout for use in pages
