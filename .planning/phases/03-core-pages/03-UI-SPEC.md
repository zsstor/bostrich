## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (from design-tokens.css) |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing (from design-tokens.css) |
| lg | 24px | Section padding (from design-tokens.css) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing (from design-tokens.css) |

Exceptions:
- Form inputs: 48px height for touch targets (A11Y-05 requirement from Phase 5)
- Grid gaps: 32px (xl) between product cards on desktop

> **Source:** Phase 1 design-tokens.css (spacing variables), CONTEXT.md D-01

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 (from design-tokens.css) |
| Label | 14px | 500 (medium) | 1.5 |
| Heading | 24px | 700 (bold) | 1.2 |
| Display | 64px | 800 (extrabold) | 1.2 |

> **Source:** Phase 1 CONTEXT.md (D-06, D-07), design-tokens.css typography scale
> **Note:** H1/H2/H3 all use 24px size. Use weight variation (H1: 800, H2: 700, H3: 600) for hierarchy while maintaining 4-size constraint.

## Notes for Planner

### Page Layout Patterns

**Home Page:**
- Single-column centered hero (CONTEXT.md D-01)
- Featured products grid: 3 columns on desktop (grid-cols-1 md:grid-cols-3) (CONTEXT.md specific ideas)

**About Page:**
- Text-only layout, no illustrations (CONTEXT.md D-06)
- Single headline + paragraph structure (CONTEXT.md D-04)

**Pricing Page:**
- 7 product sections stacked vertically (CONTEXT.md D-07)
- Each section has pricing table with 4 tiers (Free/Basic/Business/Enterprise)
- Business tier highlighted as "Recommended" (CONTEXT.md D-09)

**Contact Page:**
- Enterprise inquiry form with 4 fields (Name, Email, Company, Message) (CONTEXT.md D-11)
- Success message replaces form on submission (CONTEXT.md D-12)

### Component Inventory from Phase 2

- **Header.astro**: Desktop + mobile navigation, active page highlighting (indigo-600)
- **Footer.astro**: Multi-column grid (grid-cols-1 md:grid-cols-3), social icons
- **Layout.astro**: Base layout with semantic HTML, container pattern (max-w-7xl mx-auto px-4 sm:px-6 lg:px-8)
